# CLI News App (Python)

A command-line based News Application built using Python that allows users to fetch real-time news using the NewsAPI.

---

## Features

- Fetch Top Headlines
- Filter news by Country
- Filter by Category (Business, Sports, Technology, etc.)
- Search news using Keywords
- Fetch articles within a Date Range
- CLI-based interactive menu
- Proper error handling (HTTP & input validation)

---

## Tech Stack

- Python
- Requests Library
- NewsAPI
- PyCountry
- Datetime
